# Автоматизация для iPhone

Набор для связки **Apple Shortcuts («Команды»)** с простым API. С iPhone можно:

- получать **утренний брифинг** (дата, время, совет дня);
- **сохранять быстрые заметки** голосом или текстом;
- отправлять **webhook-события** из любых автоматизаций.

## Быстрый старт

### 1. Запуск сервера

```bash
cd iphone-automation
chmod +x run.sh
./run.sh
```

Сервер поднимется на `http://127.0.0.1:8000`. Документация: `http://127.0.0.1:8000/docs`.

Для доступа с iPhone в той же Wi‑Fi сети используйте IP компьютера, например `http://192.168.1.42:8000`.

### 2. (Опционально) Токен безопасности

```bash
cp .env.example .env
# отредактируйте API_TOKEN
export API_TOKEN=ваш-секретный-токен
```

В «Командах» добавьте заголовок: `Authorization: Bearer ваш-секретный-токен`.

## Настройка на iPhone

### Команда «Утренний брифинг»

1. Откройте **Команды** → **+** → новая команда.
2. Добавьте действие **«Получить содержимое URL»**:
   - URL: `http://ВАШ_IP:8000/api/brief`
   - Метод: GET
3. Добавьте **«Получить значение словаря»** → ключ `spoken`.
4. Добавьте **«Произнести текст»** с полученным значением.
5. (Опционально) **«Показать уведомление»** с ключом `tip`.

Запускайте вручную или через **Автоматизацию** → «Время суток» → 08:00.

### Команда «Быстрая заметка»

1. Добавьте **«Диктовать текст»** или **«Спросить ввод»**.
2. Добавьте **«Получить содержимое URL»**:
   - URL: `http://ВАШ_IP:8000/api/notes`
   - Метод: POST
   - Заголовки: `Content-Type: application/json`
   - (Если есть токен) `Authorization: Bearer ваш-токен`
   - Тело запроса (JSON):

```json
{
  "text": "Текст из предыдущего шага",
  "tags": ["iphone"]
}
```

3. Добавьте **«Показать уведомление»** → «Заметка сохранена».

Удобно повесить на виджет или кнопку Action Button (iPhone 15 Pro+).

### Webhook из любой автоматизации

Когда вы приезжаете домой, подключаетесь к зарядке и т.д. — отправляйте POST:

- URL: `http://ВАШ_IP:8000/api/webhook`
- Тело:

```json
{
  "event": "arrived_home",
  "data": {"battery": 45}
}
```

События можно смотреть: `GET /api/webhooks?event=arrived_home`.

## Идеи автоматизаций без сервера (только iPhone)

Эти сценарии работают в **Команды → Автоматизация** без этого API:

| Триггер | Действие |
|---------|----------|
| Подключение к зарядке вечером | Включить «Не беспокоить», поставить будильник |
| Прибытие домой (геолокация) | Отправить «Я дома» в Telegram/iMessage |
| NFC-метка у входа | Включить Wi‑Fi, открыть умный дом |
| Время 22:00 | Включить режим сна, уменьшить яркость |
| Открытие приложения | Таймер фокуса на 25 минут |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/health` | Проверка работы |
| GET | `/api/brief` | Брифинг дня |
| POST | `/api/notes` | Сохранить заметку |
| GET | `/api/notes` | Список заметок |
| POST | `/api/webhook` | Принять событие |
| GET | `/api/webhooks` | История событий |

## Деплой в интернет

Чтобы iPhone мог обращаться к API из любой сети, разверните на [Railway](https://railway.app), [Render](https://render.com) или VPS:

```bash
pip install -r requirements.txt
uvicorn server.main:app --host 0.0.0.0 --port $PORT
```

Установите переменные `API_TOKEN` и `DATA_DIR`.
